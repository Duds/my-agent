# Workflow: Daily Briefing

1. **Gather Data**: 
    - Retrieve Calendar events for the day.
    - Summarize unread high-priority GMail messages.
    - Check "Wealth Concept" tracker for any updates.
2. **Synthesize**: Create a Markdown summary focusing on "What needs your attention" and "What we are working on."
3. **Health Check**: Add a small AuDHD coaching tip or check-in (e.g., "Remember to stay hydrated and focus on the top 1 task today").
4. **Deliver**: Send via Telegram adapter.
